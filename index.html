    IPv6 プレフィックス / アドレスの取得方法

    1. ルーターや HGW の「IPv6 接続状態」「WAN 状態」「MAP-E 状態」などの画面を開きます。
    2. グローバル IPv6 アドレス、または委譲された IPv6 プレフィックス（/56 や /64）を探します。
    3. `fe80::` で始まるリンクローカルアドレスではなく、`2` または `3` で始まるグローバル IPv6 を入力してください。
    4. プレフィックスが表示されている場合はそのまま、アドレスだけ表示されている場合はその IPv6 アドレスを入力してください。

    PC から確認する場合の例:

    Windows: ipconfig
macOS / Linux: ip -6 addr
macOS / Linux: ifconfig

    自前ルーター運用時は WAN 側の IPv6 ではなく、MAP-E の CE 側に割り当てられているグローバル IPv6 /56 または /64 を使うと判定しやすいです。
